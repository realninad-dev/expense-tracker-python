# Expense Tracker

## Description

The Expense Tracker is a Command Line Interface (CLI) application developed using Python. It helps users record and manage their daily expenses efficiently. The application allows users to add, view, search, and delete expense records while automatically calculating the total amount spent. All expense data is stored in a JSON file, ensuring that records are preserved even after the program is closed.

---

## Features

- Add new expenses
- View all expense records
- Search expenses by category
- Delete expense records
- Calculate total expenses
- Store data permanently using JSON
